# Data_projects_TripleTen
Projects I worked on while attending the Business Intelligence Analytics TripleTen Bootcamp.

| Project Number | Title | Description |
| :-----------: | ----------- | ----------- |
| 1 | [Fresh Beats: Music Streaming Analysis](Sprint_0_README.md) | Analyzed user behavior and genre performance data for a music streaming platform. Selected insights and visualizations to produce a stakeholder-ready Project Status Report addressing artist promotion, user conversion, and genre-based content recommendations. |
| 2 | [Manhattan Airbnb Market Analysis](Sprint_1_README.md) | Analyzed NYC Airbnb listing data to advise a client on the best neighborhoods and property types to invest in for vacation rentals in Manhattan using pivot tables, data cleaning, and revenue modeling. |
| 3 | [E-Commerce Business Analytics](Sprint_3_README.md) | Transformed raw e-commerce transaction logs into a 3-stage conversion funnel and a cohort-based customer retention analysis using advanced Google Sheets techniques. |
| 4 | [Superstore Profitability vs. Returns](Sprint_4_README.md) | Built an interactive Tableau dashboard analyzing the relationship between profitability and product returns to identify which categories, regions, and segments drive profit or loss. |
| 5 | [Superstore Returns Analysis](Sprint_5_README.md) | Developed a 9-chapter Tableau story examining the root causes of returned orders across product categories, customer behavior, geography, and seasonality. |
| 6 | [Shopify App Store Analysis](Sprint_6_README.md) | Analyzed 7,341 Shopify apps using Power BI to uncover what drives ratings, review volume, and developer engagement across the platform. |
| 7 | [Zomato Customer Analytics Dashboard](Sprint_7_Final_README.md) | Capstone project using Power BI to segment Zomato's customer base via RFM analysis and demographic segmentation to identify high-value customers and re-engagement opportunities. |
| + | [From Reps to Results — Collaborative Project](Collab_Project_README.md) | A self-directed team project analyzing the top 50 exercises by muscle group, difficulty, equipment use, and calorie burn — delivering two data-backed beginner workout plans. |
